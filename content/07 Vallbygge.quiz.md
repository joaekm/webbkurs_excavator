### Fråga: Varför måste det organiska lagret schaktas bort under hela vallen ner till fast mineraljord?

- [x] Organisk jord bär ingen vall — den pressas ihop ojämnt, glider vid blötväder och sätter sig i åratal, så vallen spricker
- [ ] För att spara massor; det organiska kan annars användas i vallens kärna
- [ ] Bara av utseendeskäl; det organiska lagret bär egentligen bra
- [ ] Det behöver inte schaktas bort om vallen packas hårt ovanpå det

### Fråga: Hur packas vallen på det här projektet, när ingen padda används?

- [x] I tunna lager (ca 20–30 cm lös jord, tunnare i lerig jord) med överlappande bandöverfarter över hela kronbredden, ett lager i taget
- [ ] Genom att tippa upp alla massor på en gång och köra över högen några gånger
- [ ] Med skopan enbart; bandkörning packar inte
- [ ] Packning behövs inte om massorna är av mineraljord

### Fråga: Vad är körbarhetstestet, och vad visar det?

- [x] Maskinen kör över vallen skogsbryn–äng och tillbaka; inget kvarstående spår och ingen svikt betyder att packningen räcker
- [ ] Att man mäter krönets bredd med tumstock
- [ ] Att vallen är hög nog att köra kompakttraktor över, oavsett packning
- [ ] Ett prov på att leran är tillräckligt blöt

### Fråga: Varför är återfyllnaden kring utloppsröret vallens kritiska punkt?

- [x] Slarvig packning där ger piping — vatten följer rörets utsida, drar med finmaterial och gröper ur vallen inifrån tills den kan brista
- [ ] Röret rostar om jorden ligger för tätt an
- [ ] Det spelar ingen roll; bandkörningen packar ändå den delen
- [ ] Bara för att det ser bättre ut med jämn återfyllnad

### Fråga: Vilka åtgärder hör till en korrekt återfyllnad kring utloppsröret? (flera rätta)

- [x] Tät lera packad direkt an mot röret
- [x] En lerkrage som vandringshinder runt röret
- [x] Tunna lager (ca 10–15 cm) packade med skopa och handstamp
- [ ] Löst tillbakaskyfflade massor som får sätta sig av sig själva

### Fråga: I vilken ordning görs öppningen för utloppet, och varför?

- [x] Sist, ur den färdiga och packade vallen — och all överkörningsberoende packning görs innan, eftersom maskinen inte kan passera vallen efter håltagningen
- [ ] Först, så att man kan bygga vallen tätt runt den färdiga munken
- [ ] När som helst; ordningen saknar betydelse
- [ ] Efter att erosionsskyddet lagts, så att stenen håller kanterna

### Fråga: Vad gäller för att tömma dammen och för valet munk kontra fast tröskel på den här platsen?

- [x] Dammen kan inte tömmas under grundvattenytan på självfall, så munkens reglerfördel är illusorisk och erosionsskydd och utlopp görs rätt första gången
- [ ] Munken kan alltid tömma dammen helt via bottenröret, så den är alltid bäst
- [ ] Fast tröskel gör att dammen kan tömmas snabbare än med munk
- [ ] Ett bottenrör med ventil löser tömningen oavsett grundvattennivå

### Fråga: Hur sätts krönnivå och dämningsnivå på vallen?

- [x] Krönet i ängens nivå, dämningsnivån minst 0,5 m (hellre 0,6–0,7 m) lägre som fribord, under husdräneringsrörets nivå och kontrollerat mot fixpunkterna enligt modul 06
- [ ] Krönet så högt som möjligt och dämningsnivån strax under krönet utan marginal
- [ ] Dämningsnivån över husdräneringsröret för att få mesta möjliga vatten
- [ ] Krönet högre än ängen så att maskinen inte kan köra av vallen
